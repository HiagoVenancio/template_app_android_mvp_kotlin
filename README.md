# template_app_android_mvp_kotlin
Para facilitar o começo do desenvolvimento de uma aplicação. Template de criação com uma splash screen, tela login, e validação se está logado ou não

-Ao Entrar na aplicação, abre uma SplashScreen com um delay de tempo

-Ao Terminar o delay, verifica se está logado ou não
  --Se não tiver, vai para a Tela de Login
  --Se tiver, vai para Tela Principal
